INTRODUCTION
------------

A GUI project based on Python Tikinter that shows some image effects as well as source codes.


REQUIREMENTS
------------

This module requires the following modules:

 * Desktop Windows 10
 * Python 3.10
 * Python Tkinter
 * Python Skimage
 * Python Matplotlib
 * Python Numpy

INSTALLATION
------------
 
 * install pip3 tkinter
 * install pip3 skimage
 * install pip3 matplotlib
 * install pip3 numpy
